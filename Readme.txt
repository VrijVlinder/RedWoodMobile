How to install:
1. Move the whole 'RedWoodMobile' folder into your themes directory (.../themes).
2. Access the config.php file and add RedWoodMobile to the mobile theme configuration.
$Configuration['Garden']['MobileTheme'] = 'RedWoodMobile';
3. Add the plugin Go Back Mobile to allow going back to mobile after visiting full site.
-------------------------------------------------

This style works with the CategoriesDropdown plugin.

You can change your favicons and apple-touch-icons in the default.master.php in the views folder.
